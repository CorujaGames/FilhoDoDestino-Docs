    Introdução

    Este documento descreve as ferramentas adotadas no desenvolvimento do projeto Filho do Destino, abrangendo aquelas já implementadas, em uso atual ou previstas para uso futuro. Cada ferramenta desempenha um papel crucial na execução e apresentação das tarefas realizadas pela equipe.

    A escolha adequada de ferramentas é essencial para o sucesso de projetos de software. Segundo Flores (2022), ferramentas como Microsoft Project, Planner e JIRA são fundamentais para auxiliar gestores na distribuição, priorização e monitoramento de tarefas, visando à eficiência e eficácia do projeto. Além disso, Giocondo et al. (2021) destacam que o uso de ferramentas de gestão de projetos, como SCRUM, Kanban, Microsoft Excel, Design Thinking, Azure DevOps e MS Project, contribui significativamente para a organização e acompanhamento das etapas do desenvolvimento de software.

    As ferramentas estão apresentadas na Tabela 1, acompanhadas de seus logotipos e descrições.

    Tabela 1: Ferramentas Utilizadas
    Logo Descrição
    Miro: Usado para esboçar o GDD, mapas mentais, brainstorming e organizar as ideias iniciais de mecânicas, lore e estrutura geral do jogo.
    Zenhub: Utilizado como ferramenta de gestão ágil, com Sprints, quadros de tarefas (Kanban), controle de progresso e backlog de features.
    GitHub: Utilizado para versionamento de código, hospedagem do repositório, GitHub Pages para documentação e integração com o Zenhub.
    Unity: Engine principal utilizada no desenvolvimento do jogo. Permite criação de mapas, controle da lógica de combate e integração dos assets visuais.
    Aseprite: Ferramenta de criação de sprites e animações em pixel art. Utilizada na criação dos personagens, cenários e interfaces do jogo.
    Discord: Canal de comunicação oficial da equipe. Usado para reuniões, troca de mensagens rápidas, feedback e compartilhamento de arquivos.
    Visual Studio Code: Editor de código leve, com suporte a Git e integração com Unity. Utilizado na programação de scripts do jogo.
    Neovim: Editor de código extensível e rápido. Utilizado de forma alternativa para scripts leves, manipulação de arquivos e edição remota.

    Fonte: João Vitor Merlin, 2025
